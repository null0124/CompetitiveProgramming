
## 説明
ある列の区間の和と一点更新が高速にできる  
一応抽象化をしているが二項演算は $+$ と $-$ のままなので実質数列限定  
変なのを乗せるときは segment tree をつかう  

## 使い方
構造体  

#### 宣言
BIT<$T$> bit($N$);  
$T$: 型  
$N$: 大きさ  

#### 一点更新
bit.add($i, a$)  
- $i$: 追加する index  
- $a$: 足す値  

#### 区間取得
bit.sum($l, r$)  
- $l, r$: 区間 $[l, r)$ の和を取得する。  

### 計算量
- 一点更新・区間取得  
${\rm O}(N \log N)$
